# GaoKao-Agent

高考备考 AI 助手，支持数学、语文、理综、英语四科辅导，自动记录错题和薄弱点。

## 使用方式

直接用自然语言提问，例如：
- "这道三角函数题怎么解：..."
- "帮我分析一下《赤壁赋》的主旨"
- "我的薄弱点有哪些？"
- "距离高考还有60天，帮我规划复习"

## 触发词
- 数学题、函数、方程、几何 → 数学 Agent
- 古诗、作文、文言文、阅读 → 语文 Agent
- 物理、化学、生物、力学 → 理综 Agent
- 英语、语法、英文 → 英语 Agent
- 复习计划、薄弱点、错题 → 通用 Agent

## 技术说明
基于 OpenClaw + DeepSeek API，本地运行，数据存储在 memory/ 目录。
